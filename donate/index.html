# 伺服器贊助

各位玩家好，我們是風平浪靜NagiAsu.xyz (opens new window)，感謝各位玩家們，願意遊玩本伺服器，也希望大家可以繼續遊玩及支持我們，謝謝你們。

伺服成立已久，最初伺服開設的想法是提供一個平台，給大家在休閒之餘玩得開心，至今我們仍秉持著該理想，提供玩家更好的遊玩與服務品質。

為了滿足此目標，在伺服器營運上，時常需要有一些資金支出，如:網路費、服務租用費、設備購買、玩家回饋、人員獎勵...，這些都會經過伺服方的專業評估，確保資金皆有最大化效益，能讓玩家有更好的遊玩品質。

如果你喜歡本伺服器，同時也願意且有能力實際支持我們，歡迎提供贊助幫忙我們，小小的幫助可以讓伺服器更好。

再次感謝各位玩家們的遊玩及支持！

# 🔷 須知

  • 本伺服遵守 MINECRAFT 服務條款-商業使用準則 (opens new window)，贊助後並不會獲得任何對遊戲強度或平衡有重大影響的權限或道具。
  • 此行為為贊助行為，玩家同意以贊助的名義將款項贈與伺服器，而並非雙方買賣交易。
  • 若玩家因違反遊玩規範，而遭伺服方解除遊玩契約，款項將無法退還；已永久解除遊玩契約的玩家，也無法利用贊助來重新讓伺服方同意提供遊玩服務。
  • 贊助回饋可能會視情況修改，請以你贊助當下所見之回饋頁面說明為主。

# 💎 贊助方式

# 付款完成後，請主動使用客服系統與我們聯繫，來確認是否收到贊助，確保雙方的權益

  • 贊助金額將會以『實際收入』來統計，超商、信用卡、跨行轉帳等手續費，需由付款方自行承擔。
    • 因此若條件許可，建議使用銀行匯款、街口支付來贊助，可享有免手續費的優惠。
  • 若對於付款有任何不了解的地方，歡迎使用客服系統詢問!

  • 本回饋可能會視情況修改，請以贊助當下你所看見的回饋說明為主。

# 贊助回饋表

  • 贊助為累積制，過往已贊助的金額會累積上去

# 100元以上

  • 可進入『主伺服(三)、設施服(三)』遊玩。

# 200元以上

  • 最多可邀請2位無贊助的玩家加入『主伺服(三)、設施服(三)』。

# 300元以上

  • 可使用1個自訂名字顏色(16色)，贊助完成後自行從16色選一個顏色『Minecraft 16色 (opens new window)』

# 500元以上

  • 可使用1個自訂名字顏色(RGB色碼)，贊助完成後自行調配一個顏色做使用『自己RGB配顏色 (opens new window)』
  • 可擁有 7個 傳送記錄點(/home)
  • 傳送冷卻降低至 10秒

# 1000元以上

  • 可擁有 10個 傳送記錄點(/home)
  • 傳送冷卻降低至 5秒

# 2000元以上

  • 可擁有 15個 傳送記錄點(/home)
    • 2500元:17個、3000元:20個、3500元:22個、4000元:25個、4500元:27個、5000元:30個
  • 傳送冷卻降低至 1秒
  • 可使用2個自訂名字顏色(RGB色碼)
  • 可獲得自訂暱稱

# 回饋說明

  • 邀請玩家加入『主伺服(三)、設施服(三)』。
    • 請先前往主三或設三，之後在遊戲內輸入指令 : /wl add <要邀請的玩家ID>，來進行白名單的設定。
    • 設定後是無法移除的，所以請慎重選擇要增加的玩家。
      • 例外情況是:若被加的玩家，因贊助獲得白名單後，將會空出邀請名額，屆時可重新進行設定。
  • 名字顏色
    • 查看Minecraft 16色顏色代碼 (opens new window)、自己RGB配顏色 (opens new window)
    • 色碼選擇完成後，請使用客服與我們聯繫，將由伺服人員為您進行設定
      • 你可以開一個單人創造，把你想要的顏色改好，打指令自己看會長什麼樣子
        /tellraw @s [{"text":"你的ID","color":"#C58181"},{"text":" > ","color":"gray"},{"text":"Hello World","color":"white"}]
        把ID改成自己的 #C58181改成你想要的顏色
    • 伺服人員為您設定完成後，請輸入/tg來套用顏色，你隨時可以選擇是否要套用名字顏色。
  • 自訂暱稱
    • 可以用來做為你的暱稱(在聊天室中取代你的名字顯示)
    • 暱稱設定需透過客服，請在贊助前使用客服與我們討論，以確保您的權益。
  • 顏色/暱稱的試用期
    • 顏色及暱稱設定完成後有72小時的試用期(從幫您設定完成後開始算起)，在試用期間，您可以自由提出是否更換目前設定的顏色或暱稱，若超過試用期，往後想要更改顏色或暱稱將酌收200元費用。
