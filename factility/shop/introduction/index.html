# 稻光商店街介紹

簡介

稻光商店街是官方提供的交易場所，提供不同類型的店舖供玩家承租。
承租者可以在自己的店鋪展示並販售各類物資或畫作，也可以向其他玩家收購資源。
官方的NPC商人也在此，有空時不妨來商店街晃晃吧 !

# 如何前往稻光商店街

  • 輸入/bs返回大廳服(如果已經在大廳可以忽略此步驟)
  • 返回大廳重生點後，向後轉可看到稻光商店街傳送門，走進去後即可傳送至稻光商店街
  • 也可以沿著大廳重生點右前方的路順著走到底喔～!!!

# 稻光商店街分布圖

總數

分為六區，各區涵蓋10-15間店鋪，總計72間店鋪

# 店長

  • 店鋪負責人，負責交租及各項事務
  • 任何店面有關事務，伺服方會聯繫店長處理

# 副店長

  • 可設置商人，不可承租店鋪
  • 僅特區有此設計，可擁有商人(依照店面規定，有專屬與共用)

# 店員

  • 僅擁有領地權
