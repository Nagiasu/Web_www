# 伺服遊玩規定 (2021-12-13)

# 當您加入伺服器遊玩，即視為您同意並遵守此規定

  • 最後更新時間 : 2021/12/13

# 任何公頻發言，請遵守 公頻規範

  • 此為獨立頁面超連結，請點擊上方連結查看

# 請勿影響其他玩家遊玩

  • 破壞建築、偷東西、騷擾、詐欺、惡意誤導、無意義LAG機...等影響他人的行為
    • 基本上，在路上看到人造建築與箱子不要任意去亂碰
  • 嚴禁以非正當管道取得畫作
    • 非正當管道係指未經作者同意逕自盜印或複製畫作

# 嚴禁使用外掛功能/遊戲BUG

  • 以下幾種模組與客戶端可使用
    • 一般模組
      • 小地圖、整理背包、光影
      • 藍圖(Litematica)、連點 不可用於活動或使用腳本運行
    • 客戶端
      • LunarClient、Badlion
    • 其餘未明載之模組或客戶端若要使用，建議事先詢問客服是否允許
  • 以下模組或功能嚴禁使用(包括但不限於)
    • 任何具外掛或透視功能的材質包/模組
    • 使用含有外掛功能之客戶端遊玩 (不論是否開啟外掛功能)
    • 飛行、自動建設、自動戰鬥、KillAura...
  • 遊戲BUG / 伺服異常 / 插件漏洞
    • 原版BUG : TNT、鐵軌、地毯複製機...
      • 認定標準 : 『遊戲機制』和『實際玩法』不一致時，以遊戲機制為主
    • 利用或從中獲取利益
  • 若有上述之情事發生，伺服人員擁有主動介入處置之權利
    • 未明載但屬本項規定所涵蓋之事項，將由伺服人員認定及處置
    • 若玩家知悉上述之情事發生，請即時利用客服通報，切勿知情不報
      • 知情不報者經伺服人員查獲時將一併進行處置

# 請勿開分身或使用VPN遊玩

  • 若有這類需求，請務必主動利用客服向伺服人員詢問或提出申請
  • 違反本項規定將依照伺服人員之認定進行處置

# 嚴禁影響伺服運作

  • 嚴禁有任何影響伺服運作之行為或預謀
    • 包含但不限於以網路或電磁紀錄等方式影響伺服運作
    • 情節嚴重之抄襲或模仿，且經比對後能客觀辨認為抄襲而非原創之內容
    • 為確保伺服運作的穩定，此類個案的處置依據將可能溯及既往
    • 必要時，伺服人員需向全體玩家公開判決依據及理由
    • 若有嚴重危害而致違法疑慮者，將不排除採取法律途徑
  • 若有上述之情事發生，伺服人員擁有主動介入處置之權利
    • 未明載但屬本項規定所涵蓋之事項，將由伺服人員認定及處置

# 處分/調解與上訴

# 處分方式

  • 違反上述伺服規定，伺服有權利對玩家進行以下的處分
    • 暫時限制玩家在遊戲聊天室/Discord發言的權利
    • 暫停或終止玩家在本伺服的遊玩權限
    • 刪除或回溯玩家在本伺服的遊玩紀錄

# 案件調解

  • 主要適用於『玩家與玩家之間的檢舉』
  • 主要由被檢舉人親自向檢舉人提出，伺服方不會主動介入
    • 被檢舉人也可由客服告知伺服人員，請伺服人員作為第三方協助處理
  • 調解期限 : 檢舉人提出檢舉後，檢舉『還原完成』前
    • 若在還原完成後才提出調解，需由伺服人員同意才可受理
  • 調解流程
    • 若玩家在檢舉後，被檢舉人主動聯繫檢舉人相關調解事宜，可暫緩檢舉處理
      • 檢舉人有拒絕調解的權力
    • 請檢舉人在檢舉頻道告知伺服人員此檢舉在調解中
    • 由雙方自行私下調解(或由伺服人員協助雙方)相關補償或處理方式
    • 若調解成功，該檢舉將可撤銷，視為此案件未曾發生過

# 案件上訴

  • 玩家權利上訴
    • 玩家可在處分確定後由客服詢問/調解/上訴
      • 上訴期限 : 判決確定後一個月內
  • 處分資料
    • 資料係指:處分依據與解釋判決
      • 帶有個人隱私之內容需消去處理
        (例如:IP / 座標 / 私人檢舉...)
    • 當事人有權利詢問與要求伺服人員提供資料
      而伺服人員有義務回覆並向當事人提供與解釋
    • 雙方皆有權利向全體玩家公開資料以接受公評

# 備註

  1. 伺服處分以帳號與IP為準，請務必保管好個人帳號(切勿外借給任何人，若發生任何狀況須自行承擔責任)，不接受任何因個人因素導致之理由作為要求解除處分之依據
  2. 若發現有玩家有違反以上規定時，請即時利用客服，切勿知情不報
  3. 若對於規定有任何疑慮或意見，請利用客服主動詢問伺服人員
