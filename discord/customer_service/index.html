# 客服系統

如果你有任何事情需要我們協助，請使用此管道與我們聯繫，我們會盡快回覆

  • 下列事務有指定的申請管道，請點擊連結查看相關說明，不要使用客服系統
    • 檢舉回報(私人檢舉仍請使用客服系統)
    • 商店街事務
    • 畫作同步
  • 若有疑問與開發建議也可以直接在伺服Discord提問與開發頻道發問

# 使用方式(Discord內)

  1. 請加入伺服Discord
  2. 請在左側頻道列表往下滑動至最底並點選客服系統(操作說明動畫 (opens new window))
  3. 請點擊按鈕點我開啟客服

  4. 完成後左側選單最下方就會出現一個你專屬的客服頻道了
  5. 請在此頻道提出你想詢問的問題/要申請的事務等，並等候伺服人員回覆訊息

# 使用方式(遊戲內)

  • 輸入/msc <訊息>將訊息發送至客服系統(若客服頻道不存在會立即建立)
    • 須將遊戲帳號綁定Discord帳號
    • 如尚未建立客服頻道會立即開啟
    • 訊息會在遊戲內及Discord中同步

# 注意事項

  • 請不要濫用客服系統
  • 請不要使用客服系統問伺服人員跟伺服完全沒關係的問題
  • 客服頻道會在使用完成後刪除頻道，若有需要請自行保存紀錄

客服紀錄公開

伺服方保有將客服紀錄公開的權利，必要時伺服方得向全體玩家公開相關紀錄
