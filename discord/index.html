  • 註冊帳號 https://discord.com/ (opens new window)(官方連結)

# 伺服Discord的邀請連結

https://discord.gg/nagiasu (opens new window)

  • 點擊連結即可加入伺服的Discord

# 獲取玩家身分組

# 如果你已經登入伺服了(建議)

  • 請參考下方的連結綁定說明進行身分綁定
    • 綁定完成即會獲得綁定玩家身分組

# 如果你還沒登入過伺服

  1. 請在新人大廳輸入你的遊戲ID，並等待伺服人員審核
  2. 通過審核後即會獲得玩家身分組
  3. 參考下方的連結綁定說明進行身分綁定

注意事項

此方法會花費較長時間等待，建議先登入伺服後直接進行綁定身分

# 連結綁定說明

將你的Discord帳號與遊戲帳號進行連結
可以使用多項同步功能(如同步聊天、於遊戲內使用客服系統等)

# 綁定方式(從未綁定過)

  1. 請先登入遊戲，以及加入伺服Discord
  2. 在遊戲內輸入指令 /link後，你會得到一串六位數字

  3. 進入伺服Discord，找到下面『🎯身分驗證』
  4. 在該頻道內直接輸入六位數字
  5. 當出現連結成功後，即完成綁定
  6. 你可以確認是否擁有玩家、綁定玩家這兩個身分組

# 綁定方式(曾經綁定過)

如果你已經綁定過，但沒有玩家、綁定玩家這兩個身分組
(可能是因為你退出過群組，或是系統錯誤等原因導致)
可以在『🎯身分驗證』(不是遊戲內) 輸入/link，系統會幫你將身分組加回來

注意事項

當透過上述兩種方式皆無法成功綁定時，請使用客服系統由伺服人員協助處理

# 解除綁定

如果你換過遊戲帳號或Discord帳號，或是個人因素等原因需要解除綁定

  • 請使用客服系統請伺服人員協助處理
