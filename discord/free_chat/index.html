# 非公開Discord頻道 說明

# 說明

  • 當你想聊些不適合在公頻上聊的內容可以來這邊聊
    • 在Discord裡面的任一個頻道輸入/free指令以獲得權限
  • 非公開頻道不適用於公開聊天規範，請自行評估是否加入
    • 可能包含令人不適或不適合未成年閱覽的內容
    • 除另有規定，伺服人員將不介入任何衝突與爭議
  • 當你輸入/free獲取權限後，即視為同意本頻道的相關聲明

# 非公開頻道列表

# 🌌自由的風平

  • 這是個自由的聊天室，可以盡情暢談，不會受到公頻規範的限制
    • 本頻道有同步到遊戲內，訊息發送、接收的管理方式請詳見下方
  • 請自行拿捏尺度，這裡不限制爭吵相關的行為(不受公頻規範管制，伺服方不會介入)。
  • 請不要在這裡發表R18、獵奇的內容，請到『🔞可以色色』發表，也請不要嚴重洗頻
    • 違反本項內容，伺服人員有權力進行處置

# 🔞可以色色

  • 拿來發表、討論R18、成人內容的地方
    • 本頻道有同步到遊戲內，但僅限接收訊息
  • 嚴禁以任何形式發表有關三次元兒童色情、描述血腥、過度暴力的內容
  • 若您的內容明顯會令人不適的內容，請註記暴雷並加上警語
    • 若為文字內容，請在開頭加上警語，並將內文前後加上||反黑處理
    • 違反本項內容，伺服人員有權力進行處置

# 同步遊戲內聊天室

  • 上述頻道『🌌自由的風平』、『🔞可以色色』，都可以在伺服器內聊天室接收到訊息。
  • 你可以在遊戲內發送訊息至『🌌自由的風平』。
  1. 請先為您的Discord帳號進行綁定綁定說明
  2. 在你要加入的Discord頻道，輸入 /join
  3. 加入完成後，會出現此訊息
  4. 進入伺服內，輸入指令/c
    • 點擊第二排的羊毛，可以選擇是否要收到此頻道的訊息。
    • 點擊第一排的圖示，可以選擇你的聊天室訊息，要發送到哪個頻道。 而第三排的圖示為你目前的發送聊天室要送到哪裡。
    • 草地是『💻同步頻道』，終界眼是『🌌自由的風平』，牛奶桶是『🔞可以色色』 (『🔞可以色色』不能設定為發送訊息的頻道，只能接收訊息)

# 常見設定方式 (輸入/c設定切換)

  • 訊息發送至『💻同步頻道』，接收訊息『💻同步頻道』、『🌌自由的風平』、『🔞可以色色』。
  • 訊息發送至『🌌自由的風平』，接收訊息『🌌自由的風平』、『🔞可以色色』。
    (意味著看不到任何『💻同步頻道』的訊息)
  • 訊息發送至『💻同步頻道』，接收訊息『💻同步頻道』。
    (意味著看不到另外兩個頻道的訊息)
