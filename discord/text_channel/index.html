# Discord文字頻道

傳送文字訊息的頻道，下面詳列各頻道功能與介紹

# 規範與公告

# 👜公共頻道規範

  • 伺服的公共頻道規範，所有於公開頻道的發言請遵守本規定

# 📢伺服日誌

  • 伺服的重大更新、活動、公告等資訊都會發布在這裡
  • 請玩家密切關注此頻道的訊息，以確保了解伺服的更新資訊

# 🔨維護與更新

  • 伺服的小更新、維護、停機等資訊都會發布在這裡
  • 請玩家密切關注此頻道的訊息，以確保了解伺服的維護資訊

# 公開聊天室

# 🌅聊天室

  • 不限主題聊天，平時的聊天請多加善用這裡

# 🔎風平休憩區

  • 論壇式的聊天頻道，平時的聊天請多加善用這裡

# 同步頻道

# 💻公共頻道

  • 此頻道的訊息會與伺服內聊天室雙向同步
  • 需綁定遊戲帳號才可在此頻道發言連結綁定
  • 此頻道的所有發言，皆視為在遊戲內說話，請遵守伺服規定

# MC伺服事務處理

# 🚔檢舉回報

  • 若要檢舉玩家，請使用此頻道
  • 檢舉方式與格式請參考檢舉破壞
  • 若要調解請直接在該案件貼文中進行，或利用客服系統進行
  • 非案件相關人士請勿在此聊天

# 🧬提問與開發

  • 若遊玩時有遇到任何問題，可在此發問(也可使用客服系統)
  • 有任何對伺服的想法或意見也可在此提出，鼓勵玩家多多參與
  • 請不要出現"伺服內部無關的事情/與伺服無關的問題/無意義且無法幫助的回答"

# 🏪商店街與標案

  • 有關商店街及官方標案的資訊會於此宣布
  • 商店列表 : https://shop.nagiasu.xyz/ (opens new window)
  • 有關商店的事務申請請參考事務申請

# 🎨伺服圖分享

  • 歡迎大家在這邊貼出在本伺服遊玩的圖片
    • 請勿張貼非本伺服的圖片
  • 請盡量不要在這邊聊天

# 🧩大廳與美術

  • 大廳與美術館開發用的討論頻道
  • 非相關工作人員請不要在這裡聊天

# MC紀錄

# 🏹處分紀錄

  • 存放玩家被暫時/永久停止提供遊玩服務的紀錄
  • 存放玩家被暫時/永久限制公共頻道發言的紀錄

# 🎪商店事務

  • 商店街事務申請表單的紀錄
  • 當我們處理完成後，會標記表情符號表示處理完成
  • 有關商店街事務的申請請參考商店街事務

# 💵換錢處理

  • 商人換錢申請表單的紀錄
  • 當我們處理完成後，會標記表情符號表示處理完成
  • 有關商人換錢的申請請參考商店街事務

# 🌈畫作同步

  • 畫作同步申請表單的紀錄
  • 當我們處理完成後，會標記表情符號表示處理完成
  • 有關畫作同步的申請請參考畫作同步

# 🪐上下線紀錄

  • 存放玩家的上下線紀錄

# 🌍dc自動訊息

  • 存放玩家進入伺服Discord時的系統訊息

# 🎯身分綁定

  • 連結伺服角色與Discord帳號

# 客服系統

# 📫客服系統
