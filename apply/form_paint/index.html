# 同步說明

# 申請流程

  • 首先，請將你的畫作在製圖台利用玻璃片鎖定好，以及在鐵砧依照規則命名好
    • 圖畫命名規則橫列-直行請務按照規則命名，否則會退件處理
      • 例:星爆氣流斬1-1/星爆氣流斬1-2/星爆氣流斬2-1/星爆氣流斬2-2
  • 接著，前往原始畫作所在分流的畫作同步區，將要同步的畫作擺放進木桶內，並拿告示牌鎖好
    • 不要亂放其他東西 放進去的東西都不會還你
  • 放好後，請填寫表單，提交同步申請
    • 當處理完成後會在表單上打上表情符號，代表處理完成
  • 在同步完成之前，請先不要將地圖攜帶跨服，這會造成損壞!

# 取得自己的畫作

  • 可以輸入/maplist買自己的畫作(一張地圖5塊)
  • 如果你在同步前有自己備份一套畫作的話，可以對著你留存的畫作輸入/mapconvert即可完成轉換

# 畫作同步區地點

  • 主一 : 美術館進去後馬上右轉，樓梯的下面
  • 主二 : 商店街進去後馬上左轉，下樓梯後向右迴轉
  • 主三 : 重生點的前往美術館的門背面

# 注意事項

  • 請務必按照申請流程進行準備，以加快同步作業
    • 未按照說明處理畫作有可能會被退件，屆時請重新申請
  • 僅受理主一、主二及主三的同步申請，請務必將畫作蓋在前列分流的主世界或終界
  • 申請作業需一定時間，請玩家耐心等候處理
