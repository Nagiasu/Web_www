# 活動協助申請

# 介紹

我們鼓勵玩家們舉辦任何公開或私人的活動，促進玩家間的交流與互動，這些都能對本伺服帶來良好的效益。
若玩家有想法或行動，想要舉辦這種類型的活動，認為需要由伺服方出面協助時，可以與我們聯繫取得相關協助，本篇文將會說明你該如何進行。

若有任何第三方社群或商業合作，建議可直接聯繫本伺服Discord Discord ID：miunapa

# 有什麼是我們可以幫上忙的

  • 活動宣傳
    • 不限定於本伺服或本遊戲，若想協助宣傳第三方遊戲也可協助，我們非常歡迎玩家舉辦各式不同種類的活動
  • 伺服人員的現場協助、監督、主持、參與等人工作業
  • 物品獎勵的美化 : 更改物品名字(彩色字)/增加物品敘述 500元一件
  • 臨時的領地劃設，避免場地遭受惡意破壞
  • 臨時傳送點設置，供玩家可以直接性的傳送
  • 協助提供專用場地或伺服
  • 功能或系統的協助開發與設定使用
  • 任何討論中可能需要，且我們有辦法完成之協助

# 申請方式

  1. 請加入伺服Discord
  2. 進入『#提問與開發』頻道，接著使用討論串功能建立"公開"討論串
  3. 討論串標題請打上你的活動名稱，建立完成後，請在討論串內提出你的活動，若為外部網站連結，請確保該網站的資料是可以另存下載的。
  4. 最後，在該討論串內直接tag服主MiunaPA，並等待回應。

# 申請建議提供的相關資料

伺服方是協助玩家籌備活動或提供資源的角色，實際您需要什麼具體的需求，請務必在活動資料中詳細說明你的內容，越詳細越好，以方便本伺服可以快速理解，並從中協助您的活動籌備。
下方的項目僅為舉例內容，實際還是要依照你的想法去撰寫與說明

# 活動名稱與主旨(必需)

  • 請"簡短"說明本活動企劃的名稱與主要內容

# 活動執行時間與地點(必需)

  • 請提供本活動預計會於何時執行與地點
  • 時間
    • 一個指定時間或一段期間、特殊節慶日子等
  • 地點
    • 若為遊戲內，請附上分流、世界、座標、該地點的名稱(方便後續稱呼)
      • 若該地點為多人居住的社區或他人的建築，請務必事先取得場地所有權人同意
    • 若並非本伺服或本遊戲，請敘述為何種遊戲，並詳細說明該如何前往或參加
      • 也請詳細敘述該遊戲的主要內容

# 報名或參加條件

說明一下此活動的主要面向為何

  • 公開活動 : 任何玩家都可以參加
  • 私人活動 : 指定玩家、須滿足某種條件或報名才可參加(例如社區活動)...

# 任何你需要的幫助及詳細規劃

  • 多多益善，你需要什麼幫助，就詳細說出來，我們才知道該怎麼協助你

# 物品獎勵的美化

更改物品名字(彩色字)/增加物品敘述，請提供相關文本及格式化代碼顏色。 考量到特殊物品在玩家間具有其獨特的價值，因此這部分會有一些條件。

  • 公開與私人活動皆可申請特殊物品
  • 一件物品的價格為500元，且物品需自備，伺服方僅提供刻字與美化服務
  • 物品會強制寫上活動名稱與活動日期
  • 物品附魔外觀功能已開放給所有玩家使用，若您有需要此服務，請依照目前的規定價格支付費用

# 申請注意事項

# 請在企劃執行時間『至少』前14日繳交活動

  • 請考量到企劃的討論與審核需要時間，若逾期將不受理
  • 企劃方也需保留一定時間，以配合企劃的討論
  • 由於伺服方本身也有許多的規劃日程，請盡可能提早或保留更多的備案日期

# 我們需要有你提供的活動企劃，才能讓伺服方知道你需要什麼

  • 提出活動企劃讓伺服方知道你想做什麼很重要，沒有企劃的話，縱使你完成了所有準備工作，我們依然沒辦法幫上忙，希望大家能互相配合，依照正規流程申請，我們都會盡力去協助玩家的

# 某些情況下，活動企劃可能無法通過

  • 我們很鼓勵玩家提出企劃，也會盡量去幫忙協助企劃通過與執行，但請考量到伺服方也有一些營運上的整體決策，希望大家能互相理解，我們會盡可能讓大家的企劃都能通過，但也不排除有一些情況，是我們沒辦法審核通過的
  • 伺服方同意企劃後，也代表企劃方對此企劃需負責，而伺服方也需負起同意後的連帶責任

# 活動企劃補助

由於籌備活動需要花費大量的精力與遊戲資源，伺服會視活動企劃評估您是否可以獲得補助，如果您很需要伺服方進行相關補助的話，請提供以下說明。

# 能增進本伺服的效益

  • 請盡可能的敘述說本活動企劃若能成功執行，對本伺服能帶來的效益
    • 不論是私人或公眾效益都可以，總之盡可能說明
  • 同時也請思考或描述，本活動企劃對全伺服的公平性與合理性
    • 我們希望伺服方對大家都是盡可能公平的，這部分也請留意
