# 建築調整服務

提供玩家針對建築的調整與拆除

  • 畫作建築的付費搬遷、旋轉
  • 畫作建築的免費拆除
  • 人造建築的付費拆除

# 申請限制

# 申請人身分

  • 限由建造者本人申請調整服務
    • 若由建造者授權代為申請拆除者，請附上相關證明
    • 若為共同持有、建造的畫作/建築，請自行協調後由一人代表申請

# 畫作調整

  • 僅提供同分流且同世界內的畫作調整
    • 調整前後的畫作位置需為同分流且同世界

# 畫作拆除

  • 僅受理畫作建築的拆除
  • 與畫作相鄰的少量輔助建築可以一併申請拆除，不另外收費

# 建築拆除

  • 僅受理人造建築的拆除
  • 不受理任何自然生成結構、自然地景的拆除
    • 自然生成結構：村莊、叢林神廟、地牢等
    • 自然地景：山丘、峽谷、海床等

# 收費標準

服務類別 收費方式
畫作調整 一張地圖(128x128)大小2500元，單次(框選)收費最高上限30000元
畫作拆除 單純拆除畫作(及少量相鄰輔助建築)不需支付任何費用
建築拆除 每10000格(長x寬x高)大小2000元，單次(框選)收費最高上限20000元

# 作業時間

  • 申請通過後將由服主進行調整作業，作業時間需1-28天不等
  • 若在作業過程中發生任何問題，伺服方將會立即主動告知申請人

# 前置作業

# 畫作調整

# 確認要調整的範圍

  • 決定要調整的範圍，並記下對角線兩點座標
    • 請注意座標不要給錯
    • 可以使用磚頭來右鍵方塊查看位置

# 確認目的地座標

  • 決定移動的目的地，並記下對角線兩點座標
    • 請注意座標不要給錯
    • 目的地的對角線兩點座標的相對位置需與原座標的相對位置一致
      • 這需要一點空間概念，如果你看不懂這句話的話，可以在客服請我們協助您確認

# 確認是否需要旋轉

  • 確認畫作是否需要旋轉，並確認旋轉角度
    • 旋轉角度以90度為單位，並註記順時針或逆時針旋轉
      • 如果你看不懂這句話的話，可以在客服請我們協助您確認

# 畫作/建築拆除

# 確認要拆除的範圍

  • 決定要拆除的範圍，並記下對角線兩點座標
    • 請注意座標不要給錯
    • 可以使用磚頭來右鍵方塊查看位置

# 申請方式

  • 請利用客服系統申請，並照下列格式提供申請資料
    • 伺服人員會依照你提供的申請資料與你討論相關細節，如申請限制、預估費用等D

# 確認調整結果

# 調整結果通知

  • 調整作業完成後伺服方會馬上通知
  • 收到通知後請盡快到作業現場確認調整結果
    • 請務必親自確認調整結果以避免造成權益損失

# 確認調整結果

  • 確認調整完的位置、角度
    • 若有任何調整的問題或與申請資料不符的狀況，請立即主動提出
    • 若未於2天內確認即視為接受調整結果
  • 確定調整結果是否需要修正或還原
    • 若為玩家的資訊填寫錯誤導致調整錯誤(而不是伺服人員的作業錯誤)，有可能需支付額外費用
    • 請於2天內提出申請，逾時需重新進行案件申請

# 繳費

  • 在以上所有作業完成後，我們會跟你說最終金額是多少
  • 支付方式：/money pay Shop <金額>
  • 繳費完成後，請上傳轉帳紀錄(截圖)到客服，以利伺服人員進行確認

# 注意事項

  • 案件申請是否成立由伺服人員認定之，申請時伺服人員會在客服跟你討論
  • 畫作/建築拆除皆不會退還任何材料，若無法接受請勿提出申請
  • 申請期間請隨時注意並盡快回覆客服訊息，以節省作業時間
