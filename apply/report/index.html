# 檢舉回報

向伺服人員回報並尋求協助的管道
當你今天家被破壞、東西被偷、生物被殺等
只要是其他玩家影響你、造成你的權益受損都可以提出檢舉

# 相關連結

加入伺服Discord

  • 伺服器的Discord，要先加入才有辦法進行檢舉回報

客服系統

  • 須先加入伺服Discord，你不想透漏案件內容時會用到

# 受理項目

  • 建築、設施被破壞
  • 物品被偷
  • 生物被殺
  • 盜印畫作
  • 被其他玩家影響遊玩
    • 副本服僅受理外掛相關案件，其餘包含搶怪、殺人皆不受理
  • 有玩家做出違反遊玩規範的行為

# 檢舉流程

# 需要準備的資訊

被檢舉人的ID：有時候你貼上的圖片會有好幾個ID在上面，讓我們知道你要檢舉誰
分流：伺服有好幾個分流，讓我們知道要去哪個分流處理
事由：你檢舉他的原因，像是破壞、偷竊、外掛等，必要時可以寫詳細點
圖片：方塊紀錄查詢的截圖，需完整截圖並涵蓋座標
　　　當範圍很大時請多貼幾張圖片，貴重物品一定要貼(如烽火台、特殊物品等)
代檢舉：當你今天是幫別人檢舉時請務必註明，並註明是幫誰檢舉，且你所附上的圖片必須是你自己的查詢紀錄
影片(非必要)：檢舉外掛時，附上影片檔會讓伺服人員比較好判定
不須復原(非必要)：處理檢舉時都會同時進行復原作業，如果不需要可以註明
備註(非必要)：有需要伺服人員特別注意的地方可以寫出來，但請盡量簡短說明

# 檢舉方式

  1. 打開伺服Discord找到檢舉回報頻道
  2. 撰寫新貼文，以"文字"打上被檢舉人的ID/分流/事由等，並附上相關證據圖片(附上之圖片需為貼文者本人之查詢紀錄)
  3. 點開貼文編輯頁面裡的查看較多資訊，選擇案件對應的分流標籤後建立貼文
  4. 等待伺服人員處理(暫停/解除被檢舉人的遊玩契約、還原作業)
    處理完成後伺服人員會在你檢舉的那則貼文加上"處理完成"的標籤
  5. 檢查還原作業有沒有遺漏，有的話請在貼文內說明哪裡沒復原到(一樣要附上分流及座標等資訊)

私人檢舉

如果你不想讓別人知道你的案件內容(座標等資訊)
請使用客服系統，並在客服頻道內以相同的流程進行檢舉

# 注意事項

  • 受理項目的說明無法涵蓋所有情況，案件是否受理由伺服人員認定之
  • 檢舉資訊不完整時會要求補件，未補齊前伺服人員得不受理檢舉
  • 代檢舉時請務必以文字註明是代檢舉的案件，並註明是幫誰檢舉
    • 請謹慎提出檢舉，以避免造成當事人、被檢舉人及伺服人員的麻煩
      • 必要時伺服人員會不受理案件或進行處置
  • 請不要濫用檢舉功能，必要時伺服人員會進行處置
    • 濫用檢舉係指無意義檢舉、報復性檢舉等狀況，由伺服人員認定之
