# 2021/10/24 案件報告

案件當事人：
　　玩家 aquarter45
違反規定：
現行2021/10/24公告之伺服規定：「嚴禁有任何影響伺服運作之行為或預謀。」
　　2021/08/21公告之伺服規定：「請勿影響其他玩家遊玩。」
　　2021/07/31公告之伺服規定：「請勿影響別人遊玩(例如：破壞建築、偷東西、騷擾玩家...)。」
　　2020/08/15公告之伺服規定：「不要影響別人遊玩(例如：破壞建築、偷東西、騷擾玩家)。」
　　以上各版本伺服規定之詳細內容可於伺服規定網頁中檢閱。
處分方式：
　　永久封鎖玩家 aquarter45 遊玩本伺服器之權利
備註：
　　1.為確保案件雙方之權益，本次案件之經過及處分理由在此公開供全體玩家檢視，以接受公評。
　　2.本次案件之處分依據有溯及既往之疑慮，經伺服人員討論後決定在本次案件報告中進行釋疑。

案件經過：
2021/07/26　玩家 aquarter45 曾於伺服頻道中發表自己創建了類似本伺服器的遊玩環境。
2021/09/27　經玩家通報及伺服人員查證前述事件之真實性後，本伺服器發布聲明，呼籲各位玩家不要有私自模仿或抄襲本伺服器私有設定之行為發生，並在玩家調解區提供玩家 aquarter45 模仿本伺服器私有設定之證據。
2021/10/21　有玩家透過Discord客服系統匿名通報玩家 aquarter45 近日有欲影響本伺服器運作之計畫與意圖，經過徵得該匿名通報之玩家同意，公開不含有個人資料之證據，同時經過伺服人員內部討論後，決定將對此事件作出對策。
2021/10/24　伺服器更新伺服規定，主要為修正文本敘述方式，並明文規定出：「嚴禁有任何影響伺服運作之行為或預謀。」。本伺服器依據2020/08/15起各版本之伺服規定，認定玩家 aquarter45 之行為皆違反過去各版本之規定，並決定依據前述之事實及證據對玩家 aquarter45 做出永久封鎖之處分，並公告本次案件報告。

處分理由：
　　玩家 aquarter45 欲影響伺服運作所採用之方式據了解主要為ddos及洗頻，前者將可能癱瘓玩家對本伺服器之連線，造成全體玩家皆無法遊玩本伺服器；後者則會影響所有Minecraft及Discord在線玩家使用聊天室聊天之權利。經過伺服人員討論後，認定其預謀將會嚴重影響全體玩家遊玩之權利及遊戲體驗，因此對其做出永久封鎖之處分。

釋疑：
　　若依據2021/10/24公告之伺服規定：「為確保伺服運作的穩定，此類個案的處置依據將可能溯及既往。」，案件經過所列出之各項事實及證據將在處分當下會構成溯及既往。但根據各版本之伺服規定，玩家 aquarter45 之行為已違反過去所公告之規定，其中最早可適用於2020/08/15公告之伺服規定，因此本次案件並不構成溯及既往，特此釋疑。

伺服聲明：
　　本伺服器之網路架構有採取ddos防護措施，以提升伺服器網路環境之穩定性以及確保全體玩家的遊玩權益。同時本伺服之帳款採獨立戶頭之方式運作，各項財務收支帳目在伺服官網上皆有公開供全體玩家檢閱。且所有贊助款項皆用於伺服各項軟硬體等相關支出，並不會有挪做他用之情形發生，請全體玩家放心遊玩本伺服。最後，在此呼籲全體玩家切勿做出任何影響其他玩家或伺服器權益的舉動。
