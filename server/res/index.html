# 資源服 res

採集資源 定期洗白

  • 不建議在資源服蓋長期性建築(洗白前1週會公告)
    • 建議前往主伺服(sv)或設施服(mh)

# 前往方式

  • 從大廳服(Lobby)的分流傳送門前往
  • 使用/home或/back傳送到位於該分流的紀錄點

# 分流設定

# 基本設定

  • 地圖沒有防爆
  • 防噴裝、死亡扣除部分經驗
    • 死亡時扣除5等
      • 玩家經驗等級在10等以下時不扣除經驗

# 特殊設定

  • 沒有鎖箱/鎖門
    • 請自行保管好重要資產
  • 不會記錄在地獄的TNT破壞紀錄
    • 請勿將資產長時間置於地獄
  • 唯一可以召喚終界龍的分流
    • 請勿將終界主島占為己有
  • 唯一會生成夜魅的分流
  • 唯一會生成魚的分流
  • 唯一伏聆觸媒會蔓延的分流
  • 唯一可以挖掘地獄的遠古遺骸的分流
  • 擊殺生物有一定機率掉落生物頭顱
    • 詳細掉落機率請見連結 (opens new window)(伺服GitHub，英文)
  • 在資源服擊殺終界龍時，將會給予最終擊殺者一個鞘翅
    • 若該玩家背包無空位時，鞘翅會出現在最終擊殺者的位置上
    • 無法判定最終擊殺者時，鞘翅會生成在終界本島中間-返回傳送門的柱子上(生成時系統會強制清除該位置的方塊)

# 切換世界/分流方式

# 切換世界

  • 前往地獄請自行找地方蓋地獄門
  • 前往終界請輸入/spawn回到分流重生點後由傳送門前往

# 切換分流

  • 前往大廳服(Lobby)
    • 輸入bs後點選指南針前往
  • 前往其他分流 商店街 美術館
    • 請切換至大廳服(Lobby)後由各分流傳送門前往
