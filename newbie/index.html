# 分流介紹

這邊簡單的整理了伺服器各個分流的介紹，詳細的說明請參閱網站右上角各分流介紹

  • 大廳服：伺服器的大廳，可以通往各個分流，未來將會有美術館及商店街
    • 路上有一些特別的造景和跑酷，有興趣的話可以四處逛逛
  • 主伺服：建築蓋家及生存的地方
    • 生怪調降、關閉生怪磚功能
    • 要蓋大型牧場及設施請前往設施服
  • 設施服：建造紅石機關、物資塔及大型設施等的地方
    • 無法繁殖綿羊，請使用綿羊卷軸提升單隻綿羊的產量
  • 資源服：採集資源的地方
    • 沒有防爆、沒有鎖箱/鎖門
    • 唯一會生成夜魅、終界龍及可以挖掘遠古遺骸的分流
  • 副本服：打怪賺錢的地方
    • 獨立分流，詳細請參考副本服 簡介

# 進入伺服

你剛加入伺服的時候會出現在這邊
先把遊玩規範跟這份教學看完後再往前走吧

再來你會出現在這裡，直接往前走就可以找到主伺服的傳送NPC

接著對NPC右鍵就可以進去主伺服，你可以選擇要在主一或主二生存(主三是贊助/推薦限定)

# 找地方蓋家

進入主一或主二後，你會出現在分流重生點，對前方的告示牌右鍵就可以隨機傳送(圖片上有箭頭標示)

傳送完後再繼續往外跑個2000格以上就可以蓋家了，越遠越不容易有人來破壞跟偷你東西
(如果傳送後卡住了可以輸入/spawn，回到重生點點擊告示牌重新傳送)

當你選好地方後，可以先輸入/sethome <家的名稱>紀錄home點，然後就可以準備去資源服採集資源了
(無贊助的玩家會有5個home點可以使用，如果要刪除home點可以輸入/delhome <家的名稱>)

# 採集資源

首先輸入/bs開啟伺服幫助選單，點選上方的指南針回到大廳服(一開始選分流的那個地方)

然後輸入/spawn回到重生點，向後轉180度就可以看到資源服的傳送NPC，一樣右鍵他就可以進去了

一樣點選前方的告示牌就可以隨機傳送，找好地方就可以採集資源了
(如果傳送後覺得位置不好可以輸入/spawn，回到重生點點擊告示牌重新傳送)

採集完資源後就可以輸入/home <家的名稱>，回到剛剛選好要蓋家的地方開始蓋你的家
接下來就跟一般生存一樣了，如果有需要蓋大型牧場或設施請務必前往設施服

# 常用功能與指令

這裡整理了一些常用指令，詳細的說明請參考網站左側選單的各個章節

# 開啟新手教學連結

  • /rule在聊天欄顯示官網及新手教學頁面連結(/rules也是同樣效果)

# PVP

  • /pvp開啟/關閉PVP

# 伺服幫助選單

  • /bs開啟伺服幫助選單

# 聊天室

  • /tell <玩家ID> <文字>發送密語給他人(/m , /msg 也是同樣效果)
  • /r <文字>密語回覆上一個發送你密語的人

# 傳送/設定紀錄點

# 傳送

  • /spawn返回至伺服重生點
  • /back回去上一個傳送點
  • /tpa <玩家ID>請求傳送至對方那邊 你-->對方
  • /tpahere <玩家ID>請求"對方"傳送至你這邊 對方-->你

# 設定紀錄點

  • /sethome <家的名稱>設定家的位置
  • /home <家的名稱>傳送回家
  • /delhome <家的名稱>刪除現有的家
  • /homelist查看你擁有的家及位置

# 方塊紀錄查詢

  • /co i開啟/關閉查詢功能
  • /co near查詢你所在位置附近的事件(半徑5格內)

# 鎖箱子/門

  • 對箱子或門拿告示牌右鍵就可以上鎖，這樣別人就不能去動它
    • 資源服沒有鎖箱子/門

# 金錢系統

  • /money查詢自己有多少錢
  • /money pay <玩家ID> <金額>給別人錢
  • /money lookup <玩家ID> <頁數>查看交易紀錄
