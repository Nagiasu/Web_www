# 基本酒類

  • 材料以粗體藍字表示者 請於副本任務 (opens new window)中取得

# Wheatbeer 小麥啤酒

  • Skunky Wheatbeer/Wheatbeer/Fine Wheatbeer
  • 材料：
    • 小麥 3-6
  • 烹煮：2-5分鐘
  • 貯存：樺木桶
  • 難度：1

# Beer 啤酒

  • Skunky Beer/Beer/Fine Beer
  • 材料：
    • 小麥 3-6
  • 烹煮：4-8分鐘
  • 貯存：任意木桶
  • 難度：1

# Darkbeer 黑啤酒
