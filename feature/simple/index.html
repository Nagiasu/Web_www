# 簡易小功能

# 骰子

  • /roll <點數最小值> <點數最大值> <重複次數>依照設定擲出骰子
    • 很簡單的隨機數獲取，可以用來賭博或當抽籤用
    • 使用此功能需消耗青金石，請手持青金石才可操作
    • 可自由設定點數最小值/最大值和重複次數
      • 皆需輸入整數
      • 若未輸入最小值與最大值，預設範圍為1-6
        • 若需要設定範圍，請同時輸入最小值與最大值
      • 最小值需>=1且最大值需<=2147483647
      • 重複次數需<=10，輸出時會顯示數值加總
    • 當你骰下去後 你半徑20格內的玩家會收到此訊息
    • 將游標移動至骰子結果上，可查看該次擲骰的詳細訊息
    • 骰子指令原始碼

# 指令示範

  • /roll
    • 骰出 1顆 1-6 的骰子
  • /roll 1 20
    • 骰出 1顆 1-20 的骰子
  • /roll 1 6 3
    • 骰出 3顆 1-6 的骰子 (消耗3顆青金石)

# 開啟官網及新手教學連結

  • /rule在聊天欄顯示官網及新手教學頁面連結(/rules也是同樣效果)

# PVP

  • /pvp開啟/關閉PVP
    • 開啟後須冷卻30秒才可關閉
      • 冷卻期間若有攻擊玩家會重製計時
    • 各分流設定不同步
  • /pvp status查看PVP狀態

# 顯示/隱藏玩家頭上的ID

  • /hidename顯示/隱藏玩家頭上的ID
    • 每次隱藏會消耗20元遊戲幣
    • 各分流設定不同步
      • 分流重開後會重置設定，恢復為顯示

# 主三/設三白名單

說明

此功能為贊助玩家限定，同時只能在主三或設三使用，詳細請參考贊助頁面說明

  • /wl add <玩家ID>將指定玩家加入你的白名單列表
  • /wl查看你的白名單朋友列表

# 排除幽靈方塊

  • /ghost排除半徑16內的幽靈方塊
    • 當你遇到幽靈方塊(Ghost Block)的狀況時，這個指令可以幫助你更新你周遭16格球體的方塊，以解決此狀況。

# 界伏盒開啟

  • 界伏盒可以在背包直接對它按右鍵開啟

# 移除界伏盒的NBT標籤及名字

  • /removeboxtag移除背包內所有空盒的標籤及名字
    • 用於商店街交易，避免因標籤問題使商人無法辨識，而無法進行交易

# 盔甲架保護

  • 將盔甲架保護後，只有你可以修改及對盔甲架造成傷害(但不防其他玩家射箭/縱火)
    • 設定方式：手持命名牌對盔甲架右鍵(會消耗)，即可完成保護
    • 備註：若你後來更改ID，請透過射箭的方式去移除盔甲架

# 資訊查看

# 查詢線上玩家

  • /glist在聊天欄顯示上線玩家列表
  • /find查詢指定玩家的在線狀態
    • 在線會顯示在哪個分流，不在會顯示不在線上

# 鐵砧懲罰等級(repaircost)

  • /repaircost查詢手上物品的鐵砧懲罰等級(repaircost)

# 紅磚頭

  • 對蜂巢點右鍵，可查看蜂箱蜜蜂數量與座標
  • 對海龜點右鍵，可查看海龜重生點在哪(讓海龜下蛋會用到)
  • 對村民點右鍵，可查看床與工作點的座標
  • 對可馴服的生物右鍵，可查看該生物的主人
  • 對可繁殖的生物右鍵，可查看該生物的成年所需時間或交配冷卻時間
  • 對告示牌右鍵，可查看該告示牌的文字內容
  • 對任意方塊點右鍵，可以查看該方塊的座標位置

# 六角恐龍桶

  • 用水桶撈六角恐龍，可查看顏色及生長階段
