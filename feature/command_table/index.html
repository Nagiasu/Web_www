# 指令列表

這裡整理了所有指令跟最基本的說明，提供玩家快速查詢指令

注意事項

指令的詳細使用方式請點擊標題參考個別頁面的詳細說明

指令 說明
/bs 開啟伺服幫助選單

# 聊天室

  • 點我前往說明頁面
指令 說明
/tell <玩家ID> <文字> 發送密語給他人(/m , /msg 也是同樣效果)
/r <文字> 密語回覆上一個發送你密語的人
/ignore <玩家ID> 將指定玩家禁言，重新輸入即可解除禁言
/ignore 查看被你禁言的玩家
/c 開啟頻道管理介面
/ch chat <頻道代號> 設定發話頻道
/lm 調整上下線訊息顯示方式
/mcs <訊息內容> 將訊息傳送至Discord客服頻道(須綁定Discord)
@<玩家ID> 在遊戲內tag玩家(對方須綁定Discord)
:<表情符號ID>: 在遊戲內傳送伺服Discord表情符號
/mail send <玩家ID> <內容> 在遊戲內寄信給其他玩家
[item] [i] [inv] [hand] 展示主手物品
[offhand] 展示副手物品
[helmet] [head] [hat] 展示頭盔物品
[chestplate] [body] 展示胸甲物品
[Leggings] [Legging] [leg] 展示褲子物品
[Boots] [boot] 展示鞋子物品
/tg 開啟名稱管理介面
/tguse 使用稱號兌換卷

# 傳送/設定記錄點

  • 點我前往說明頁面
指令 說明
/spawn 返回至分流重生點
/back 回去上一個傳送點
/tpa <玩家ID> 請求傳送至對方那邊
/tpahere <玩家ID> 請求"對方"傳送至你這邊
/tpaccept 接受最近的傳送請求
/tpaccept <玩家ID> 接受指定玩家的傳送請求
/tpdeny 拒絕最近的傳送請求
/tpdeny <玩家ID> 拒絕指定玩家的傳送請求
/sethome <家的名稱> 設定家的位置
/home <家的名稱> 傳送回家
/delhome <家的名稱> 刪除現有的家
/homelist 查看你擁有的家及位置

/co l 進階搜尋功能，使用方式詳見官方介紹 (opens new window)(英文)
