# 商店街支票系統

全新的商店街交易模式，結合貨幣系統與商人系統。

# 指令使用 /cq

  • /cq 打開商店街支票購買介面
  • /cq help 取得幫助
  • /cq <面額> 取得該面額的商店街支票
    • 如果主手上持有商店街支票，則會更新手上的商店街支票面額。
  • /cq sell 售出持有的支票
    • 請將要售出的支票拿在主手上
    • 本功能目前僅能在『主伺服(二)』使用，其餘分流不可使用。
    • 一天只能售出『64張支票』(換回64元押金)，跨日重置額度。

# 支票樣式與功能說明

  • 交易時才會向您的戶頭(/money)扣錢，而不是在購買支票時扣錢。
    • 詳細使用說明和交易流程會在下方敘述。
  • 支票本身沒有實際價值，不可用來作為貨幣使用
    • 僅有押金1元，將於使用時折抵消費金額。
  • 任何人皆可取得任意數量的支票，而上面的面額也能自由更改。
  • 『1元、10元、100元、1000元』的面額會用染料呈現，而其他的面額會用紙來呈現(如:8763元)。
  • 交易時，店家不會收到實際的支票物品，而是直接匯入帳戶餘額內。
    • 店家可輸入/money lookup查看交易紀錄。
    • 補貨箱不需要留格子給支票，可以多放幾樣東西。
  • 支票不能使用鐵砧改名。

# 買家 購買物品交易方式

  1. 假設買家看到一個 需要100元支票的東西

  2. 買家如果一張支票都沒有 輸入/cq 可以打開買商店街支票的交易介面。

  3. 點擊粉色染料 購買一張100元的支票，此時戶頭將會扣除1元押金。

  4. 向商人交易，此時你的戶頭會扣除(100-1)元。
    一張支票的押金是1元，在此筆交易中你用掉了『一張』商店街支票，而此商品的價格是100元，因此你的戶頭會扣除99元，來完成此筆交易。 店家將會收到100元的收入。
    備註 : 如果你戶頭的餘額不足99元，交易將會失敗。

  5. 買家可以在/money lookup看到此筆交易紀錄。

  6. 店家也能透過/money lookup看到此筆交易紀錄。

# 多張支票的面額扣除舉例

  • 上圖是100元x11 + 10元x8 = 這個東西售價1180元。
  • 購買時買家會扣除1161元(19元是押金)，店家會收到1180元。

# 店家 販售物品價格設定與操作流程

以下圖舉例，店家請將自己需要設定的面額支票放在2格交換物品當中

  • 可自由搭配支票的面額與數量，也可以一格支票一格其他物品。
  • 兩格的支票面額與數量相乘後加總，即為該物品的價格。
  • 如圖上白色界伏盒的價格為 100元x2張 + 10元x4張，即為240元，而賣家在交易時就需要準備這些支票來交易。
  • 該物品也可設定為1張240元支票，或24張10元...，由店家自行決定。
  • 設定多少張、多少面額的支票，即為賣家交易時需要準備多少張對應面額支票來完成此交易。
    • 請盡量以賣家能快速完成交易的設定為主。
    • 為了方便買家辨識，建議除非有必要，否則盡量使用『1元、10元、100元、1000元』的面額和數量來設定價格。
    • 其他自訂面額的外觀都是紙，買家會需要滑鼠移動上去才可辨認價格，也會需要先關閉交易介面後自行輸入指令取得該面額支票後，才能完成交易。
  • 當交易完成後，店家不會收到實際的支票物品，而會直接將收益匯入至店家的帳戶餘額內。
    • 店家可輸入/money lookup查看交易紀錄、誰在什麼時候買了什麼。
    • 補貨箱不需要留格子給支票了，可以多放幾樣東西。

# 店家收購物品、玩家賣物品給店家

# 玩家販售物品給店家

  • 若店家有收購物品，買家將會看到此交易畫面。
    交易結果會是一顆雪球(標示用 非實際物品)，上面寫著店家收購此物品的價格。
  • 請將店家需要的物品放在交易介面上，然後點擊雪球。
  • 此筆收購交易完成，玩家將可獲得金錢，而店家的帳戶將會扣除對應的金額。
  • 雪球僅為介面標示用途，在交易完成後玩家不會實際拿到。

# 店家設定

  • 請將第一格(交易結果物品)放上商店街支票，支票的數量即為你想要收購的價格，只能放一張支票。
    • 不要放100元x2張，請放一張200元支票。
    • 不能同時在收購物與交易物都放入商店街支票。
  • 如下圖為收購價10元(放了一張10元商店街支票)。
  • 接著，根據你剛剛所放的商店街支票面額數量，在你的箱子放入該面額的商店街支票若干張，這會用來控制你所想要收購的數量。
    • 如下圖，在補貨箱放了3張商店街支票10元，代表你最多想收購三次交易。
    • 每完成一筆交易將會減少一張支票，若箱子無該面額的支票將無法進行收購。
  • 當收購時，玩家(賣方)將會獲得10元，而店家(你)將會扣除9元。
    • 少的這1元是商店街支票的押金退回。

# 舊有商店街代幣換回金錢

  • 請至商店街事務提出申請，任何玩家皆可使用
  • 面額無限制，每人每週最多申請一次
  • 兌換開放至2023/02/28，逾期關閉
