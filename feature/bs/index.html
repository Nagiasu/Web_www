# 伺服幫助選單/bs

關於伺服的常用指令幫助、市場系統以及回到大廳服，都可使用此選單直接查看或執行

# 主選單

  • 標題：會顯示你所在的分流
    • 目前大廳服暫時沒有/bs選單
    • 副本服的/bs選單與其他分流獨立
  • 羅盤：點擊可以回到大廳服
  • 燈籠：點擊會在聊天欄顯示伺服官網連結
  • 玩家頭顱：會顯示你的ID
  • 鐘：點擊會在聊天欄顯示伺服Discord邀請連結
  • 綠寶石：開啟交易市場
  • 書：點擊會在聊天欄顯示指令列表連結
  • 幽靈眼淚：開啟頻道系統選單
  • 命名牌：開啟名稱系統選單
  • 講台：開啟商店街選單
    • 限主一/主二
      • 會隨新商店街進度有所調整
