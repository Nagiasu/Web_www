# 特效系統

提供玩家多元遊玩視覺體驗

# 使用方式 /pp

  • 輸入/pp開啟特效介面，即可點擊特效套用
    • 點擊屏障可清除特效
  • 副本服無法使用特效
  • 目前大廳無法更改使用的特效，如果要更換自身所掛的特效或取消使用，請至生存服(主伺服/設施/資源)
