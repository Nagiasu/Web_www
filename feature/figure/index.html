# 人物動作/裝備/頭顱

# 坐下、躺下、趴下

  • /sit坐下

    • 也可以對著樓梯/地毯/半磚按右鍵
      • 如果你不希望按右鍵坐下 請輸入/sittoggle

  • /lay躺下

  • /crawl趴下

# 戴帽子

  • /hat把東西戴在頭上
    • 手持要戴的東西，輸入指令即可戴上
      • 手上只能有一個物品，超過會無法戴上
    • 若要脫下來，直接打開背包將物品拿下即可
    • 物品或方塊都可以，呈現的樣式各有不同，請自行體驗
      • 可輸入F5切換視角查看

# 脫下綁定裝

  • /takeoff <部件名稱>脫下綁定的裝備
    • 如果你不小心穿上有綁定詛咒的裝備，可輸入指令脫下此裝備
      • 主手不能拿東西
    • 脫下頭盔 /takeoff helmet
    • 脫下胸甲 /takeoff chestplate
    • 脫下護腿 /takeoff leggings
    • 脫下靴子 /takeoff boots

# 頭顱獲取

# 玩家頭顱 獲取條件

  • 使用指定武器來擊殺玩家
    • 手持獄髓斧頭
    • 帶有鋒利五+絲綢附魔
    • 被擊殺的玩家的經驗值至少要有3等以上
    • 達成上述條件後，該玩家死亡時即會掉落他的頭顱
  • 被擊殺掉落頭顱之玩家，將會扣除1+1的經驗等級
    • 1等為因PVP而死亡的經驗懲罰，未超過十等時則不會扣除
    • 1等為因掉頭而造成的經驗扣除
    • 範例
      • 13等被符合條件之武器擊殺而死，此時會掉落頭顱，剩下11等
      • 9等被符合條件之武器擊殺而死，此時會掉落頭顱，剩下8等
      • 2等被符合條件之武器擊殺而死，什麼都不會掉
    • 掉落玩家頭顱設定只適用於主伺服/設施服，資源服不掉落玩家頭顱

# 資源服 生物頭顱

  • 玩家在資源服擊殺生物有一定機率掉落生物頭顱
    • 詳細掉落機率請見連結 (opens new window)(伺服GitHub，英文)
