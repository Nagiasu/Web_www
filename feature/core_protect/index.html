# 查詢方塊建造/破壞紀錄

如果家裡被破壞可以使用這些指令查詢是誰破壞的
並將分流座標及記錄 回報至Discord🚔檢舉回報並等候管理員處理

  • /co i開啟/關閉查詢功能
    • 對方塊點左鍵：查詢該方塊位置的建造/破壞紀錄
    • 對方塊點右鍵：查詢相鄰方塊位置的建造/破壞紀錄、容器的物品流向
    • 查詢功能關閉前無法正常建造/破壞方塊
  • /co near查詢你所在位置附近的事件
    • 查詢範圍為玩家所在位置半徑五公尺(格)內
    • 可查詢方塊建造/破壞、丟出/撿起物品、殺害生物等事件
  • /co l進階搜尋功能
    • 使用方式詳見官方介紹 (opens new window)(英文)
    • 通常用不太到，前兩個指令就夠用了
    • 玩家使用u:參數時僅有查詢自己的權限，無法查詢別人的紀錄
  • /co l <page>指定頁數查詢
    • 在使用/co i、/co near及/co l後，可進一步使用此指令查詢指定頁數的紀錄
    • 玩家使用/co l <page>查詢頁數時，僅可查詢9999頁以內的資料

# 紀錄儲存期限

分流 時間
主伺服 永不清除
設施服(一)(二) 60天
設施服(三) 30天
資源服 14天
  • 檢舉以有辦法查詢到破壞者為主，若紀錄過於久遠以至於資料清除，無法檢舉與還原
  • 請玩家定期巡視自己的建築與資產完整性
