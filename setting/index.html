# 基本設定

  • 沒有領地及保護區
    • 東西被偷被破壞可以透過Discord進行檢舉回報
      • 我們會盡快幫你處理，並復原你被破壞、被偷的東西
  • 防噴裝、死亡扣除部分經驗
    • 除了副本服的特殊情況
    • 死亡時扣除5等
      • 玩家經驗等級在10等以下時不扣除經驗
  • 防爆及防燒
    • 除了資源服
  • 有PVP
    • 預設為關閉，請自行輸入/pvp開啟/關閉
  • 只有資源服可以挖掘地獄的遠古遺骸；會生成魚類、夜魅及終界龍
